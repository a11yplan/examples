Installation

npm
yarn
pnpm

Installation mit npm

npm install @a11yplan/scanner

Installation mit yarn

yarn add @a11yplan/scanner

Installation mit pnpm

pnpm add @a11yplan/scanner

Systemanforderungen

  • Node.js 18 oder höher
  • npm 9+ oder yarn 1.22+
  • Mindestens 512 MB RAM

    • Globale Installation

    npm install -g @a11yplan/scanner
