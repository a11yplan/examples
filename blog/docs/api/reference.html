Vollständige API Referenz

Scan-Optionen

Parameter Typ Standard Beschreibung
url string - Erforderlich. Die zu scannende URL.
wcagVersion string "2.2" WCAG Version (2.0, 2.1, 2.2)
level string "AA" Konformitätsstufe (A, AA, AAA)
bestPractices boolean false Best Practices einbeziehen
experimental boolean false Experimentelle Regeln einbeziehen

Crawler-Optionen

Parameter Typ Standard Beschreibung
maxDepth number 1 Maximale Crawl-Tiefe
maxPages number 10 Maximale Seitenanzahl
include string[] [] URL-Muster zum Einschließen
exclude string[] [] URL-Muster zum Ausschließen

Response-Format

Die API gibt Daten im JSON-Format zurück.

{
  "id": "scan_123",
  "status": "completed",
  "violations": [...],
  "passes": [...],
  "incomplete": [...]
}
