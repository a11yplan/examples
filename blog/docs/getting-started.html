Erste Schritte

Hinweis: Diese Anleitung setzt Node.js 18+ voraus.

1. Installation

Installieren Sie das Scanner-Paket über npm:

npm install @a11yplan/scanner

Hinweis: Alternativ können Sie yarn oder pnpm verwenden.

Optionale Abhängigkeiten

Für erweiterte Funktionen installieren Sie zusätzliche Pakete.
