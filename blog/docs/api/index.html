API Referenz

Diese Seite ist auf Tiefe 2 (/docs/api/). Vollständige REST API Dokumentation.

Authentifizierung

Alle API-Anfragen erfordern einen API-Key im Header:

Authorization: Bearer YOUR_API_KEY

Endpunkte

POST /api/v1/scan

Startet einen neuen Accessibility-Scan.

url string Die zu scannende URL
level string WCAG Level (A, AA, AAA)

GET /api/v1/results/{id}

Ruft die Ergebnisse eines Scans ab.

GET /api/v1/reports

Listet alle verfügbaren Berichte auf.
