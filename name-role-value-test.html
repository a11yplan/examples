WCAG 4.1.2 Requirements:

  • Name: Accessible name (aria-label, aria-labelledby, or associated label)
  • Role: Component role (button, checkbox, link, etc.)
  • Value: Current value/state (aria-checked, aria-pressed, value attribute)
  • States/Properties: Must be programmatically settable
  • Use native HTML controls when possible
  • Custom controls need ARIA attributes
