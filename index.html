About This Repository

This collection contains machine-readable, AI-parsable test samples for validating accessibility scanning tools and algorithms. Each test page is designed with:

  • Structured metadata - Data attributes for automated parsing
  • Expected results - Documented violations and passes
  • Test case IDs - Following TC{category}.{number} pattern
  • JavaScript API - window.getTestMetadata() for validation
  • No external dependencies - Self-contained HTML files
