Keyboard Navigation

Tastaturnavigation ist essentiell für barrierefreie Websites.

Wichtige Tasten

  • Tab - Zum nächsten fokussierbaren Element
  • Shift + Tab - Zum vorherigen Element
  • Enter - Element aktivieren
  • Escape - Dialog oder Menü schließen

Best Practices

  • Verwenden Sie tabindex="0" oder tabindex="-1"
  • Vermeiden Sie positive tabindex-Werte
  • Stellen Sie sicher, dass alle interaktiven Elemente fokussierbar sind
