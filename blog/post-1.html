WCAG 2.2 Einführung

Die Web Content Accessibility Guidelines (WCAG) 2.2 wurden im Oktober 2023 veröffentlicht.

Neue Erfolgskriterien

WCAG 2.2 führt mehrere neue Kriterien ein:

  • 2.4.11 Focus Not Obscured (Minimum) - Level AA
  • 2.4.12 Focus Not Obscured (Enhanced) - Level AAA
  • 2.4.13 Focus Appearance - Level AAA
  • 2.5.7 Dragging Movements - Level AA
  • 2.5.8 Target Size (Minimum) - Level AA
  • 3.2.6 Consistent Help - Level A
  • 3.3.7 Redundant Entry - Level A
  • 3.3.8 Accessible Authentication (Minimum) - Level AA
  • 3.3.9 Accessible Authentication (Enhanced) - Level AAA

Implementation

Bei der Umsetzung sollten Sie schrittweise vorgehen.

Zusammenfassung

WCAG 2.2 ist ein wichtiger Schritt für die digitale Barrierefreiheit.
