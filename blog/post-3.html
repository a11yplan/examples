Keyboard Navigation implementieren

Warum ist Keyboard-Navigation wichtig?

Viele Nutzer können keine Maus verwenden und sind auf die Tastatur angewiesen.

Grundprinzipien

1. Alle interaktiven Elemente fokussierbar

Buttons, Links und Formularelemente müssen mit Tab erreichbar sein.

2. Sichtbarer Fokusindikator

Der aktuelle Fokus muss deutlich sichtbar sein.

Demo: Focus-Stile

3. Keine mouse-only Interaktionen

Klickbares Element (nur Maus)

4. Tab-Reihenfolge logisch

Link 1 (tabindex=5) | Link 2 (tabindex=3) | Link 3 (tabindex=1)

Tasten und ihre Funktionen

  • Tab: Zum nächsten Element
  • Shift+Tab: Zum vorherigen Element
  • Enter/Space: Element aktivieren
  • Escape: Dialog/Modal schließen
  • Pfeiltasten: Navigation innerhalb von Widgets
