API Referenz

Diese Seite ist auf Tiefe 2 (/docs/api/). Vollständige REST API Dokumentation.

Authentifizierung

Alle API-Anfragen erfordern einen API-Key im Header.

Endpunkte

POST /api/v1/scan

Startet einen neuen Accessibility-Scan.

Weitere Endpunkte finden Sie in der vollständigen Referenz.

Rate Limits

Die API ist auf 100 Anfragen pro Minute beschränkt.
