Detection Method

Automated detection should check:

  • Computed width and height of interactive elements
  • Include padding, border, and margin in calculations
  • For text links: line-height + padding determines minimum height
  • Spacing exception: controls can be smaller if spacing creates 24√ó24px offset
  • Flag violations when total clickable area < 24√ó24 CSS pixels

Measurement Guidelines

Include in target size calculation:

  • Content box (width √ó height)
  • Padding (all sides)
  • Border (thickness on all sides)
  • Margin (creates spacing offset)

Exceptions (Not Required to be 24√ó24px)

  • Inline text links - Within sentences/paragraphs
  • Essential controls - When target size is essential to functionality
  • User agent control - Browser default controls
  • Spacing - Smaller targets with adequate spacing between them
