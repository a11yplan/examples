Erste Schritte

Hinweis: Diese Anleitung setzt Node.js 18+ voraus.

1. Installation

Installieren Sie das Scanner-Paket über npm:

npm install @a11yplan/scanner

2. Ersten Scan durchführen

Führen Sie einen einfachen Scan durch:

npx a11yplan scan https://ihre-website.de

3. Ergebnisse analysieren

Die Ergebnisse werden in der Konsole angezeigt und als JSON-Datei gespeichert.

Ergebnis Violations Anzahl gefundener Fehler

Tipp: Verwenden Sie die --verbose Flag für detaillierte Ausgaben.
